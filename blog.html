Important FAQ :

What IS The Document Object Model (DOM)?

When a web page is loaded, the browser creates a Document Object Model of the page.

how do you select an element from the dom?

getElementById , getElementsByClassName , querySelector , getElementsByTagName

what is event delegation in the context of the dom and why is it useful?

Event delegation is a technique in JavaScript for handling events at a higher level in the DOM rather than attaching event listeners directly to individual elements.

How do you manipulate an elements attributes and style using the dom?

the onfocus event handler method is used to change the < input > element's style when the user clicks inside it.
